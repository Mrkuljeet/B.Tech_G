Class Selector (.class)

The class selector is used with a dot (.) before the name. It can be applied to multiple elements.

Example: .class-example { color: blue; }

ID Selector (#id)

The ID selector uses a hash (#) before the name and should be unique for each element.

Example: #id-example { border: 2px dashed orange; }

Descendant Selector (parent child)

The descendant selector targets elements nested inside another element.
