Understanding the Box Model

Every HTML element is treated as a box in CSS. The box model consists of: Content, Padding, Border, and Margin. Padding adds space inside the element, border surrounds it, and margin creates space outside it.

Example of Margins, Padding, and Borders

This box has a dashed orange border, 25px padding (inner space), and 30px margin (outer space). It demonstrates how these CSS properties create visually balanced layouts.
